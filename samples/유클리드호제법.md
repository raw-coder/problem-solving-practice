### 유클리드 호제법

재귀를 통해 빠르게 gcd 를 구하는 방법


```cpp
#include <stdio.h>

int gcd(int a, int b) {
    if(b == 0) return a;
    return gcd(b, a % b);
}

int main() {
    printf("%d\n", gcd(10, 50)); // 10
    printf("%d\n", gcd(12, 15)); // 3
    printf("%d\n", gcd(gcd(12, 15), 60)); // 3   
    return 0;
}

```
